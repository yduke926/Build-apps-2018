React App Boilerplate 

**Install all the node packages** 
On the root of this project run on your terminal (if you want you can do this with yarn but thats optional)
    
    npm install
    
 **Update the node packages** 
On the root of this project run on your terminal (if you want you can do this with yarn but thats optional)
    
    npm update


**Start the server**

  npm run watch
